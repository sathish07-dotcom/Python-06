Lists & Tuples in python

In Python, **lists** and **tuples** are both used to store multiple items in a single variable. While they have similarities, they have key differences in terms of mutability, syntax, and usage. Let's explore them in detail:

Lists in Python:

  • A **list** is a mutable, ordered collection of items. Lists can contain items of different data types (e.g., integers, strings, other lists, etc.), and you can modify, add, and remove items from a list after it has been created.

    • Key Characteristics:

  • Mutable: You can modify a list after itâ€™s created (e.g., add, remove, or change elements).
  • Ordered: The items in a list have a defined order, and they will retain their order as they were added.
  • Allows duplicates: Lists can contain multiple occurrences of the same value.

    • Creating a List:

  • Lists are created using square brackets `[]`.

    • my_list = [1, 2, 3, "apple", 4.5]

    Accessing Elements:

  • You can access individual elements of a list using an index, where the index starts at 0.

    • print(my_list[0]) # Output: 1 print(my_list[3]) # Output: apple

    Modifying a List:

  • Since lists are mutable, you can change an element or append new items.

    • # Modifying an element
    my_list[1] = "orange"
    print(my_list) # Output: [1, 'orange', 3, 'apple', 4.5]

    # Appending an item
    my_list.append("banana")
    print(my_list) # Output: [1, 'orange', 3, 'apple', 4.5, 'banana']

    #### Other List Methods:

    # Removing an element by value
    my_list.remove("apple")

    # Popping an element (removes the last item)
    popped_item = my_list.pop()
    print(popped_item) # Output: banana

    # Finding the index of an element
    index = my_list.index(3)
    print(index) # Output: 2

    # Sorting the list
    my_list.sort()
    print(my_list) # Output: [1, 3, 'orange', 4.5]

    Tuples in Python:

  • A **tuple** is an ordered collection of items, similar to a list, but unlike lists, tuples are **immutable**. Once a tuple is created, its elements cannot be modified (i.e., you cannot change, add, or remove elements).

    • Key Characteristics:

  • Immutable: Once a tuple is created, its elements cannot be modified.
  • Ordered: The items in a tuple have a defined order, and they will retain their order as they were added.
  • Allows duplicates: Tuples can contain multiple occurrences of the same value.

    • Creating a Tuple:

  • Tuples are created using parentheses `()`.

    • my_tuple = (1, 2, 3, "apple", 4.5)

    Accessing Elements:

  • Like lists, you can access elements of a tuple using an index.

    • print(my_tuple[0]) # Output: 1
    print(my_tuple[3]) # Output: apple

    Tuples are Immutable:

  • You cannot modify, add, or remove elements from a tuple after it is created. If you try to modify a tuple, you'll get an error.

    • # This will raise an error
    my_tuple[1] = "orange" # TypeError: 'tuple' object does not support item assignment

    Other Tuple Operations:

  • Although you cannot modify the elements of a tuple, you can perform operations like slicing and concatenation.

    • # Slicing a tuple
    sliced_tuple = my_tuple[1:4]
    print(sliced_tuple) # Output: (2, 3, 'apple')

    # Concatenating two tuples
    another_tuple = (6, 7, 8)
    new_tuple = my_tuple + another_tuple
    print(new_tuple) # Output: (1, 2, 3, 'apple', 4.5, 6, 7, 8)

    # Repeating a tuple
    repeated_tuple = my_tuple * 2
    print(repeated_tuple) # Output: (1, 2, 3, 'apple', 4.5, 1, 2, 3, 'apple', 4.5)

    Differences Between Lists and Tuples

    Feature List Tuple
    Mutability Mutable (can be changed after creation) Immutable (cannot be changed after creation)
    Syntax Defined using square brackets `[]` Defined using parentheses `()`
    Use cases Used when you need to modify data over time Used when data should remain constant
    Methods Many methods like `append()`, `remove()`, `pop()` Fewer methods, e.g., `count()`, `index()`
    Performance Slower than tuples due to mutability Faster than lists due to immutability
    Duplicates Can contain duplicate values Can contain duplicate values
